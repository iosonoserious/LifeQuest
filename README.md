# LifeQuest: Finance Simulator

پروژه ترم درس برنامه‌سازی من، یه بازی کنسولی شبیه Sims ولی با تمرکز روی مدیریت مالی شخصی.

## بازی چیه

هر ماه یه نفر (Sim) دارید که باید زندگیش رو بچرخونید: حقوق بگیرید، قبض بدید، پول بریزید توی حساب پس‌انداز یا سرمایه‌گذاری، ملک/ماشین/سهام بخرید و بفروشید. هر از گاهی هم یه اتفاق رندوم میفته (بونوس، دزدی، بیمارستان، بخت‌آزمایی...). اگه ۳ ماه پشت‌سرهم نتونید قبض بدید ورشکست می‌شید، اگه شادیتون صفر بشه هم بازی تموم میشه.

## ساختار کد

- `Person` — مشخصات سیم (انرژی، شادی، سن)
- `Job` — شغل‌ها (پارت‌تایم، فول‌تایم، فریلنسر، کارآفرین)
- `Account` — حساب‌ها (جاری، پس‌انداز، سرمایه‌گذاری)
- `Expense` — هزینه‌ها (ثابت، متغیر، اضطراری)
- `Asset` — دارایی‌ها (ملک، ماشین، سهام)
- `Event` — اتفاقات رندوم زندگی
- `GameEngine` — موتور اصلی بازی و منوی کنسول
- `SaveManager` — ذخیره و بارگذاری بازی روی فایل

همه‌شون (به‌جز GameEngine و SaveManager) یه کلاس abstract پایه دارن و چندتا زیرکلاس که ازش ارث‌بری کردن. اصول OOP (encapsulation, abstraction, inheritance, polymorphism) همه‌جای کد هستن، مخصوصاً جایی که GameEngine روی لیست `unique_ptr<Account>` یا `unique_ptr<Asset>` لوپ می‌زنه و virtual function صدا می‌کنه بدون اینکه بدونه نوع دقیقش چیه.

## اجرا

```
make
./lifequest
```

روی ویندوز با MSYS2 (`pacman -S mingw-w64-ucrt-x86_64-gcc make`) هم اجرا می‌شه. روی اندروید با Termux هم میشه، فقط باید بنویسید `make CXX=clang++` چون Termux پیش‌فرض clang داره نه g++.
